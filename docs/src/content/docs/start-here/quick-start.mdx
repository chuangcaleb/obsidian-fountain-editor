## üî® Usage

### üèÅ Quickstart

1. üì• [Open the installation prompt in Obsidian](https://obsidian.md/plugins?id=fountain-editor) (recommended)
   1. How to install Obsidian plugins (beginner) [Link ‚Üí](https://help.obsidian.md/Extending+Obsidian/Community+plugins)
   2. Manual installation (advanced) [Link ‚Üí](#-manual-installation)
2. üìÑ Mark a markdown note for fountain formatting:
   1. Use a `.fountain.md` extension
   2. Add a `#fountain` tag to the frontmatter metadata Properties
   3. add `cssclasses: fountain` to the frontmatter metadata Properties.
3. üé® You will get **automatic syntax highlighting/styling according to the [Fountain syntax](https://fountain.io/syntax/)**! Try copy-pasting this:

```markdown
---
cssclasses:
  - fountain
---

# Opening Hook

> THE STINK LINK <

FADE TO:

EXT. PUBLIC RESTROOM - DAY

A POLICE OFFICER steps into a dingy restroom. She winces.

POLICE OFFICER (V.O.)
I _smell_ a rat...

= Use Fountain's Synopsis to escape into [[regular markdown]]! #so-cool

%%
You can use **Obsidian comment blocks** to escape into markdown too!

> [!tip] Strip these Obsidian comments out with the Obsidian Longform plugin!

%%
```

1. üÜï You may install directly with [BRAT](https://github.com/TfTHacker/obsidian42-brat) using the link of this repo (`https://github.com/chuangcaleb/obsidian-fountain-editor`).
